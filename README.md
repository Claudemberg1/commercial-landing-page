Commercial-landing-page-curso-com-ia
Este projeto consiste em uma Landing Page de Alta Performance, desenvolvida sob demanda para a captação de leads e conversão de vendas para o curso "Ganhe Dinheiro com Vídeos IA". A aplicação foi estruturada para oferecer uma experiência de usuário fluida, com foco total em converter o tráfego em vendas através da oferta hospedada na Hotmart.

Descrição do Projeto
A página foi planejada para suportar alto volume de tráfego (pago e orgânico). Possui uma estrutura leve, design moderno com fundo animado (Canvas), vídeos incorporados e gatilhos mentais (copy comercial) estrategicamente posicionados. O objetivo principal é transformar visitantes em alunos, guiando-os através de depoimentos, grade de conteúdo e botões de chamada para ação (CTA) com efeitos de pulso para máxima visibilidade.

Tecnologias Utilizadas
Para garantir o melhor índice de qualidade em plataformas de anúncios (Google/Meta) e carregamento instantâneo, utilizamos:

HTML5 Semântico: Para melhor indexação e acessibilidade.

CSS3 Avançado: Estilização moderna utilizando variáveis, animações de entrada (fade-in, slide-up) e design responsivo.

JavaScript (Vanilla): Implementação de fundo tecnológico animado via canvas e manipulação de eventos sem a necessidade de bibliotecas pesadas.

Firebase Hosting: Distribuição global de conteúdo com baixa latência e segurança SSL.
